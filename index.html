Trigger is a multifunctional discord bot to take care of any and all of your needs

Features
  • Secure Moderation
  • Utility
  • Fun Commands(memes and more!)
  • Server Specific Leveling
  • Optional Logging
  • Fast and Simple Reaction Roles
  • Auto Messages(Join/Leave)
    • How to get started
  • Add Trigger to your favorite servers!
  • Use the command !help
  • If you want Trigger to have a different prefix use !prefix .(or any other symbol or short phrase you may like)
  • If you run into any problems join the help server
  • Note: Some of Trigger's commands require certain discord IDs.
